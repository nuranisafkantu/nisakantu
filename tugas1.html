Perintah-perintah untuk mengupload(push) ke repository


git init untuk memulai

git status untuk melihat status

git add untuk menambahkan perubahan yang sudah dibuat

git commit -m "pesan commit atau perubahan" untuk menambahka pesan yang mewakili pekerjaan yang dilakukan

git branch -M untuk memindahkan pesan

git remote add origin https://github.com/nuranisafkantu/nisakantu.git untuk menhubungkan dengan project

git push -u origin main untuk mengupload file ke repository
